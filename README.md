## Library Management System
A full-featured library management system built using HTML, CSS, and JavaScript, with Java OOP backend logic. Members can borrow and return books, while librarians can manage the entire system.

## Features:
1- Member registeration and login
2- Browse and search books by title or author
3- Borrow and return books with due dates
4- View personal borrow history
5- Edit member profile
6- Librarian dashboard to add and remove books
7- Librarian can view borrow history for each book and member
8- Search within books, members, and borrowed books lists

## Tech Stack:
- Frontend: HTML, CSS, JavaScript
-  Backend: Java (OOP)
-  Spring Boot + database integration coming in v2

## How to Run:
Open 'index.html' in your browser, no installation needed.

## Login:
- Librarian: any ID starting with '9'
- Member: Register first then login with your membership number

## Why I Made This:
I created this project to practice web development by converting an existing Java library system into a fully functional web app. This is my first HTML, CSS, and JavaScript project, so it was a greate oppurtunity to learn how frontend development works and how it connects to backend logic. It also allowed me to see how the same system can be built in two different languages.
